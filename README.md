# Scarce web store

### 1. Dashboard

* Revenue summary
* Weekly & monthly order stats
* Growth / decline indicators
* Best seller items

### 2. Orders Management

* View all orders
* Filter by status
* Filter by date range
* Weekly order insights
* Order details view
* Update order status

### 3. Inventory Management

* Add new products
* Update product stock
* Product variations

  * Bulk create variations
  * Edit existing variations
